E-Commerce Product Catalog Platform

Adunola Mojolaoluwa Olusegun • 10/02/2026

Project Overview

  • What: Full-stack e-commerce platform for browsing and managing a product catalog
  • Purpose: React frontend + Django REST API backend
  • Key value: Filtering, sorting, pagination, admin dashboard, JWT auth
  • Scope: Product catalog, category filters, admin CRUD, cart UI, reviews

Features & Functionality

Customer: Browse, filter (category, price), sort, paginate, product cards, search bar, responsive

Admin: JWT login, create/update products, logout

UX: Loading skeletons, error states, empty states

UI Design Rationale

  • Design system: Primary blue (#1e40af), Inter font, 4px base unit
  • Layout: Fixed navbar, filter sidebar (drawer on mobile), responsive grid
  • Product cards: Hover lift, stock badges (green/yellow/red)
  • Accessibility: Focus rings, semantic structure

Technologies Used

LayerTechnology
FrontendReact 18, TypeScript, Vite
StateRedux Toolkit (RTK Query)
StylingTailwind CSS
BackendDjango 5.0, DRF
AuthJWT (Simple JWT)
API DocsSwagger / ReDoc

Best Practices & Standards

  • Frontend: Component-based, TypeScript, RTK Query caching
  • Backend: DRF conventions, indexed models, validators
  • General: Env config, CORS, API documentation
